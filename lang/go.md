### Variables

* Declare
```go
  var x int
  x = 1
  
  var y int = 1
  
  z := 1
  
  i, j := 1
```

* Variable will be assigned default value after declared

### Function

* Declare
```go
  // no return value
  func log(message string) {
  }
  
  // return one value
  func add(a int, b int) int {
  }
  
  // return two value
  func power(name string) (int, bool) {
  }
  
  // concise version
  func add(a, b int) int {
  }
```
